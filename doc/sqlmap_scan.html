8. Rekomendasi Remediasi

  1. Gunakan prepared statements / parameterized queries (PDO atau MySQLi) pada seluruh query.
  2. Terapkan input validation dan whitelist karakter pada parameter pencarian.
  3. Ganti penyimpanan password ke algoritma bcrypt / argon2 dengan salt unik.
  4. Terapkan prinsip least privilege pada akun database aplikasi (read-only bila memungkinkan).
  5. Aktifkan Web Application Firewall (WAF) sebagai lapisan pertahanan tambahan.
